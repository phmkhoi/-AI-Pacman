## Introduction

This is Pacman game - an AI Project from University of Science (HCMUS)  
This project is programmed entirely by Python language.  

## Building and running

To run the program, you must install `pygame` library.  
To build and execute the program, follow these steps:
- Step 1: Open the `terminal` or `command line`
- Step 2: Navigate to the directory containing `main.py`
- Step 3: Run this command:
```
python main.py
```
